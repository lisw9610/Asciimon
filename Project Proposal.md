## Project: Asciimon

**Team Members:** Liam Sweeney, Nguyen Pham  
**Language:** Java  
**Platform:** Spring Boot  

### Description
Our project will be to create a simple online card based fighting game inspired by card games such as Magic: the Gathering and Pokemon. User's will fight an opponent by using cards from their deck. There will be elemental advantages such that additional or reduced damage is dealt/taken based on the typing of both the user's and opponent's card. The cards themselves will be displayed using ascii art. The actual user interface will be a simple HTML one that allows for the user's to queue for matches, select their cards, and choose what actions to take in a match. 

### Tentative Features
- **Gacha system:** New cards can be drawn using currency earned from winning matches. Requires database to store the cards that each player has along with currency and other necessary records. 
- **Card progression:** Using a card in matches allows it to level up and improve its stats. Requires determining amount of XP and stat increase for each level along with storing that info for each user.
- **Items in battle:** Consumable items such as potions and buff items that can be used during matches.
- **Ranking-based matchmaking:** Match players with similar skill levels to improve fairness and encourage improvement.
