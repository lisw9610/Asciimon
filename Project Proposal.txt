Asciimon(?)
Liam Sweeney, Nguyen Pham
Language: Java, Platform: _______

Description:
Our project will be to create a simple online card based fighting game inspired by card games such as Magic the Gathering and Pokemon. User's will fight an opponent by using cards from their deck. There will be elemental advantages such that additional or reduced damage is dealt/taken based on the typing of both the user's and opponent's card. The cards themselves will be displayed using ascii art. The actual user interface will be a simple HTML one that allows for the user's to queue for matches, select their cards, and choose what actions to take in a match. 

Tentative:
-Gacha feature where new cards can be drawn using currency earned from winning matches. Would require database to store the cards that each player has along with currency and other necessary records. 
-Card experience where using a card in matches allows it to level up and improve its stats. Would require determining amount of xp and stat increase for each level along with storing that info for each user.
-Items that can be used during matches such as potions and buff items.
-Ranking system so that more skilled players get matched with those of a similar skill level. Improves fairness of matches and encourages user's to improve.
